## 4-BIT-ADDER
### Entradas:
8 entradas
### Saídas:
4 saidas
### Explicação: 
Um 4 bit-adder é um circuito que realiza a soma de dois números binários de 4 bits.

## HALF-SUBTRACTOR
### Explicaçao:
Um half-subtractor é um circuito lógico usado para subtrair dois bits. Ele tem dois bits de entrada: A e B, e duas saídas:
D (diferença): Resultado da subtração (A - B).
B (empréstimo): Indica se houve um "empréstimo" ou "borrow" durante a subtração (se B for maior que A).
